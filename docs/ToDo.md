# Ladder
## 구현 기능

- Ladder: 사다리 생성
- drawLine: 사다리 라인 그리기
- run: 사다리 게임하기

## 조건 / 제한

- 게임하는 사람 수, 사다리 층의 수
  - 1 < 사람 수
  - (사람 수 + 1) <= 사다리 층 수
- 사다리 라인 그리는 위치
  - 0 < x < 사다리층-1
  - 0 <= y < 사람수-1
  - 사다리 라인의 사전 존재 여부
